## User(firstname, lastname, email, creditcard) :

- **Firstname :**
    - mustn't be null
- **Lastname :**
    - mustn't be null
- **Email :**
  - must respect the regex

##Credit Card (number, expiration date, owner name)

- **Number :**
  - must have 18 digits
  - mustn't be null

- **Expireation Date :** 
  - must be before the system date
  - mustn't be null

- **Owner Name :** 
  - must be the same as the user's
  - mustn't be null
